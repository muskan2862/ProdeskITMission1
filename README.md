# ProdeskITMission1
Prodesk IT is a responsive digital agency landing page built using HTML, CSS (Flexbox &amp; Grid), and JavaScript. It features a sticky navbar, hero section, service cards, and footer. Includes dark mode toggle, hover animations, and mobile-friendly design, showcasing modern UI and frontend skills.
